# aqua-iq-pro
Aqua IQ Pro is an ESP32 IoT system for live bus tracking and seat booking via IP 10.109.244.222.  Features: Live Tracking: GPS, Speed, and ETA.  Smart Alerts: 500m proximity &amp; brake heat warnings.  Seat Booking: Route search and reservations.  Diagnostics: Oil quality &amp; passenger counting.  Data Logs: Instant CSV export.
